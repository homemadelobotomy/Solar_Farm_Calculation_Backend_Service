# Разработка интернет-приложений (РИП) 2025

## Расчет выработки энергии солнечной фермой

Разработанное приложение дает возможность консультантам легко и быстро создавать завки на расчет мощности с разными типами панелей и площадами покрытия, а инженерам отслеживать данные заяки и принимать решения по роасчету мощности:отклонить или завершить расчет
